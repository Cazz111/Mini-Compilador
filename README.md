# Mini-Compilador

Proyecto de un mini compilador desarrollado para la materia de **Traductores** en la carrera de **ISTE (2025-02)**.

El programa toma un pseudocódigo simple  y lo traduce a **ensamblador x86**, incluyendo análisis léxico, sintáctico y generación de código. Este proyecto sirve como práctica para entender cómo funciona el proceso completo de traducción de un lenguaje.

Versión final entregada para la materia.
